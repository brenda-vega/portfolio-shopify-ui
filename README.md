# Portfolio Shopify UI — Brenda Vega

Variante independiente del portafolio UI enfocada en vacantes de UI Designer para e-commerce, Shopify storefronts y responsive interfaces.

## Enfoque

- UI Design para e-commerce
- Shopify storefront concepts
- Product pages y collection pages
- Mobile-first responsive UI
- Product cards, banners y layouts comerciales
- Design systems y componentes reutilizables
- Interfaces orientadas a claridad y conversión

## Proyectos destacados

1. Tokopedia Mobile Adaptation
2. Dulce Felicidad Shopify Concept
3. Fidex Capital
4. Kairos / Phronesis

## Deploy

Este sitio es estático. Para GitHub Pages puede publicarse desde `main` usando la raíz del repositorio:
