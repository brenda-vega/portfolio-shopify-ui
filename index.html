CASO 01 UI Designer · E-commerce · Mobile-first · Marketplace
Tokopedia Marketplace Experience
Rediseño conceptual de experiencia e-commerce mobile-first enfocado en jerarquía visual, navegación clara y conversión.

Problema

Tokopedia presenta una alta densidad visual típica de los marketplaces: múltiples promociones, categorías y productos compitiendo simultáneamente por la atención del usuario.

Enfoque

Diseñé una experiencia más limpia y modular, reduciendo ruido visual y mejorando la jerarquía de producto mediante un UI System más consistente, navegación mobile-first y componentes orientados a conversión.

Qué hice

  • Rediseño de homepage desktop y mobile.
  • Product Detail Page (PDP).
  • Shopping Cart.
  • Checkout Flow.
  • Wishlist Experience.
  • Sistema visual consistente para cards, navegación y CTAs.
  • Optimización de spacing, tipografía y jerarquía visual.

El resultado es una experiencia e-commerce más clara, moderna y premium, manteniendo la familiaridad de Tokopedia mientras mejora la escaneabilidad, navegación y flujo de compra.

Proyecto conceptual realizado con fines de exploración UI y portafolio.

CASO 02 Shopify · E-commerce · Responsive Design
Dulce Felicidad Shopify Experience
Concepto visual para una tienda Shopify enfocada en una experiencia de compra cálida, moderna y orientada a conversión.

Problema

Muchas tiendas pequeñas de alimentos y repostería presentan interfaces saturadas o poco consistentes, dificultando la exploración de productos y reduciendo la percepción de marca.

Enfoque

Diseñé una experiencia e-commerce más limpia y emocional, utilizando layouts responsivos, jerarquía visual clara y una presentación de producto enfocada en mejorar navegación y conversión.

Qué hice

  • Homepage e-commerce.
  • Product sections y banners promocionales.
  • Collection layouts.
  • Product cards.
  • CTAs y bloques promocionales.
  • Sistema visual responsive para Shopify.

El resultado es una experiencia visual más moderna, coherente y comercial, enfocada en reforzar identidad de marca y mejorar la experiencia de compra mobile-first.

Enfoque: Shopify, Responsive Design y Conversion.
